# 🧇 WaffleMan — by ApeXhit

> The central app & website directory manager for the ApeXhit ecosystem.

WaffleMan lets you manage a unified directory of your apps and featured websites, then embed that directory as a live widget across all your platforms — one publish, everywhere updated.

## Features

- 📦 **App Grid** — Display your apps in a golden waffle grid layout
- 🌐 **Featured Sites** — Showcase external websites in a clean list
- 🚀 **One-Click Publish** — Push changes to the live API instantly
- 🔄 **Auto-Sync Embeds** — All embedded widgets update automatically
- 🎨 **Custom Icons** — Upload icons or paste image URLs
- 💻 **Embed Anywhere** — HTML snippet, React component, or direct API

## Embed Widget

```html
<div data-waffleman></div>
<script src="https://apexhit-waffleman.vercel.app/widget.js" async></script>
```
